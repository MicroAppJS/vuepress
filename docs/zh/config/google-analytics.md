# Google Analytics

配置 `Google Analytics` 参数。

## Usage

使用方式如下：

```js
// micro-app.vuepress.config.js
module.exports = {
    GAID: '' // UA-00000000-0
}
```

**参考：**

- [plugin-google-analytics](https://v1.vuepress.vuejs.org/zh/plugin/official/plugin-google-analytics.html)

## Options

### GAID

- 类型: `string`
- 默认值: `undefined`

提供 Google Analytics ID 以启用集成。
